# Overview

Multi-Droplet Wave Simulator is an interactive web application that numerically solves the **two-dimensional wave equation** using an explicit **Finite Difference Method (FDM)**.

Originally developed as a MATLAB numerical simulation, the project has been redesigned into a full-stack application consisting of:

* **Python + FastAPI** simulation backend
* **React + Vite** interactive frontend
* **WebSocket-based** real-time communication
* **NumPy** numerical computation engine

The application allows users to configure multiple simulation parameters, compare different boundary conditions, visualize wave propagation in both **2D** and **3D**, and observe how disturbances evolve over time.

# Mathematical Model

## Governing Equation

The simulator solves the two-dimensional wave equation

```text
∂²u/∂t² = c² ( ∂²u/∂x² + ∂²u/∂y² )
```

where

* **u(x,y,t)** — surface displacement
* **c** — wave propagation speed

---

## Explicit Finite Difference Discretization

The numerical solution uses an explicit second-order finite difference stencil

```text
u(n+1) =
2u(n)
− u(n−1)
+ rx²(u(i,j+1)−2u(i,j)+u(i,j−1))
+ ry²(u(i+1,j)−2u(i,j)+u(i−1,j))
```

where

```text
rx = cΔt/Δx

ry = cΔt/Δy
```

---

## CFL Stability Criterion

Numerical stability requires satisfying the Courant–Friedrichs–Lewy (CFL) condition

```text
c Δt √(1/Δx² + 1/Δy²) ≤ 1
```

A stable timestep is automatically computed as

```text
Δt = 0.4 × min(Δx,Δy)/(c√2)
```

ensuring stable time integration throughout the simulation.

---

## Gaussian Disturbance Model

Each raindrop is initialized using a Gaussian distribution

```text
u(x,y)=A exp(-r²/(2σ²))
```

where

* **A** — disturbance amplitude
* **σ** — droplet radius
* **r** — distance from the droplet center

This produces realistic circular wave fronts.

---

# Boundary Conditions

## Reflecting Boundary (Neumann)

The reflecting boundary conserves energy by forcing the normal derivative to zero.

Outgoing waves are reflected back into the simulation domain, producing standing wave patterns and long-term interference.

---

## Absorbing Boundary (Mur 2nd Order)

The absorbing boundary approximates an infinite domain.

Outgoing waves leave the computational grid with minimal reflection, allowing more realistic open-water simulations.

---

# Simulation Modes

## Mode 1 — Symmetric Ring

Multiple droplets are placed around a circular ring.

All disturbances occur simultaneously, generating complex interference patterns.

---

## Mode 2 — Sequential Center Drops

All droplets are released from the center at fixed time intervals.

This produces concentric expanding wavefronts that interact over time.

# Results & Observations

| Property                | Reflecting Boundary | Absorbing Boundary |
| ----------------------- | ------------------- | ------------------ |
| Edge Behaviour          | Waves reflect       | Waves exit         |
| Energy                  | Conserved           | Dissipates         |
| Interference            | Strong              | Minimal            |
| Physical Interpretation | Closed tank         | Open water         |

The Mur absorbing boundary provides a realistic approximation for unbounded domains while the reflecting boundary is useful for studying resonance and interference phenomena.
